# 바코드 물품 관리 시스템 구현 계획

## 📋 프로젝트 개요
기존 마진 계산기에 바코드 스캔 기능과 유통기한 관리가 가능한 물품 관리 시스템을 추가합니다.

## ✅ 작업 목록

### 1. UI/UX 구조 설계
- [ ] 탭 네비게이션 추가 (마진 계산기 ↔ 물품 관리)
- [ ] 물품 관리 섹션 레이아웃 구성
  - 바코드 입력/스캔 영역
  - 물품 정보 입력 폼
  - 물품 목록 표시 영역

### 2. 바코드 스캔 기능
- [ ] 바코드 수동 입력 필드 구현
- [ ] 카메라를 이용한 바코드 스캔 기능 (선택적)
  - QuaggaJS 또는 html5-qrcode 라이브러리 사용 고려
  - 모바일 카메라 접근 권한 처리

### 3. 물품 정보 관리
- [ ] 물품 데이터 구조 설계
  ```javascript
  {
    id: string,              // 고유 ID (timestamp 기반)
    barcode: string,         // 바코드 번호
    name: string,            // 물품명
    category: string,        // 카테고리 (선택)
    quantity: number,        // 수량
    expiryDate: string,      // 유통기한 (YYYY-MM-DD)
    registeredAt: string,    // 등록일시
    price: number,           // 가격 (선택)
    memo: string            // 메모 (선택)
  }
  ```
- [ ] 물품 입력 폼 구현
  - 바코드 (자동/수동)
  - 물품명
  - 수량
  - 유통기한 (날짜 선택)
  - 가격 (선택)
  - 메모 (선택)

### 4. 데이터 저장 및 관리
- [ ] LocalStorage를 이용한 데이터 영구 저장
- [ ] CRUD 기능 구현
  - Create: 새 물품 추가
  - Read: 물품 목록 조회
  - Update: 물품 정보 수정
  - Delete: 물품 삭제

### 5. 물품 목록 표시
- [ ] 등록된 물품 카드/리스트 뷰
- [ ] 각 물품 카드에 표시할 정보
  - 물품명
  - 바코드
  - 수량
  - 유통기한 (D-day 표시)
  - 등록일
- [ ] 정렬 기능
  - 유통기한 임박순
  - 등록일순
  - 이름순
- [ ] 검색/필터 기능
  - 물품명 검색
  - 바코드 검색
  - 유통기한 필터 (만료된 것, 7일 이내, 30일 이내)

### 6. 유통기한 알림 기능
- [ ] 유통기한 D-day 계산 및 표시
- [ ] 유통기한 임박 시각적 표시
  - 만료됨 (빨강)
  - 7일 이내 (주황)
  - 30일 이내 (노랑)
  - 그 외 (정상)
- [ ] 대시보드에 요약 정보 표시
  - 만료된 물품 수
  - 7일 이내 만료 물품 수
  - 전체 물품 수

### 7. 추가 편의 기능
- [ ] 물품 수량 증감 버튼 (+ / -)
- [ ] 바코드로 기존 물품 빠른 검색
- [ ] 엑셀/CSV 내보내기 (선택)
- [ ] 물품 사진 추가 (선택)
- [ ] 통계 페이지 (선택)
  - 카테고리별 통계
  - 월별 소비 추이

## 🎨 디자인 가이드
- 기존 마진 계산기의 디자인 시스템 유지
- 모바일 우선 반응형 디자인
- 직관적이고 사용하기 쉬운 인터페이스
- 터치 친화적인 버튼 크기

## 🔧 기술 스택
- HTML5
- CSS3 (기존 스타일 시스템 활용)
- Vanilla JavaScript
- LocalStorage (데이터 저장)
- 바코드 스캔 라이브러리 (선택: html5-qrcode 또는 QuaggaJS)

## 📱 주요 기능 흐름

### 물품 등록 플로우
1. 바코드 스캔 또는 수동 입력
2. 바코드로 기존 물품 검색
   - 있으면: 수량 추가 옵션 제공
   - 없으면: 새 물품 정보 입력
3. 물품명, 유통기한, 수량 등 입력
4. 저장 버튼 클릭
5. 목록에 추가되고 성공 메시지 표시

### 물품 조회 플로우
1. 물품 목록 탭에서 전체 목록 확인
2. 검색/필터로 원하는 물품 찾기
3. 물품 카드 클릭 시 상세 정보 및 편집 옵션 표시

## 📝 구현 우선순위
1. **Phase 1 (핵심 기능)**
   - 탭 네비게이션
   - 바코드 수동 입력
   - 기본 물품 정보 입력 (이름, 바코드, 유통기한, 수량)
   - LocalStorage 저장
   - 물품 목록 표시
   - 유통기한 D-day 표시 및 색상 구분

2. **Phase 2 (편의 기능)**
   - 바코드 카메라 스캔
   - 검색/필터 기능
   - 수량 증감 버튼
   - 물품 수정/삭제

3. **Phase 3 (고급 기능)**
   - 통계 및 대시보드
   - 엑셀 내보내기
   - 물품 사진 추가

작업은 Phase 1부터 순차적으로 진행합니다.
